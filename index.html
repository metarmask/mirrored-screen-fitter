För att använda i Chrome

  1. Klistra in chrome://flags/#enable-experimental-web-platform-features i adressfältet.
  2. Aktivera den markerade flaggan genom listan till höger om den.
  3. Klicka “Starta om nu” längst ner till höger på sidan.

Generella instruktioner

  1. Klicka en gång någonstans på sidan.
  2. Om det står att sidan frågar efter en kamera, se instruktioner för att använda i Chrome ovan.
  3. Tillåt att sidan får del av din skärm.
  4. Ändra storleken på videon genom att dra med höger musknapp.
  5. Ändra positionen på videon genom att dra med vänster musknapp.
  6. Dölj den här texten med H.
